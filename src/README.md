# Строд-Сервис Технолоджи - Цифровое управление строительством

## Описание проекта
Передовая система управления строительными услугами с интегрированным хранилищем файлов и возможностями создания отчетов.

## Архитектура
- **Frontend**: React + TypeScript
- **Backend-Python**: FastAPI
- **Backend-DotNet**: ASP.NET Core
- **Database**: PostgreSQL
- **File Storage**: MinIO или локальное хранилище

## Структура проекта
```
strodservice-tech/
├── docs/                     # документация
├── backend-python/           # Python бэкенд (FastAPI)
├── backend-dotnet/           # .NET бэкенд (ASP.NET Core)
├── frontend/                 # React + TypeScript фронтенд
├── shared/                   # общие спецификации, API-схемы
├── .gitignore
└── README.md
```

## Функциональные возможности
1. Главный экран - информация о текущих, будущих и завершенных проектах
2. ПТО - полный цикл управления проектами от контракта до сдачи
3. Коммерческий и исполнительный отделы
4. Управление материалами и складскими остатками
5. Исполнительная документация
6. Формирование отчетов
7. Интеграция с файловым хранилищем
8. **Новый раздел: ГПР (График Производства Работ)** - позволяет отслеживать планируемые и фактические объемы работ по различным типам материалов на разных объектах с детализацией по дням

## Новый раздел: ГПР (График Производства Работ)

Добавлен новый раздел "ГПР", предназначенный для заполнения исполнительным директором и начальником участка. Раздел включает:

- Основную таблицу с заказчиками, объектами и видами работ
- Плановые и фактические объемы работ по различным типам материалов
- Визуализацию прогресса выполнения работ
- Ежедневную детализацию выполнения работ
- Сводную таблицу по материалам с возможностью формирования недельных отчетов

Подробнее см. документацию в файле [docs/gpr_section.md](docs/gpr_section.md).
