# Раздел ПТО (Производство технических операций)

## Общее описание
Раздел ПТО является ключевым компонентом системы "Строд-Сервис Технолоджи" и предназначен для автоматизации и точного контроля и учета в процессе строительства.

## Раздел №1 - Список проектов
Содержит следующую информацию:
1. Наименования подрядной организации
2. Наименование объекта полностью или кратко с возможностью дальнейшего редактирования
3. Дата и номер контракта
4. Протяженность объекта
5. Предполагаемые объемы производства работ
6. Дата укладки асфальта
7. Дата начала и дата завершения наших работ
8. Дополнительные комментарии

## Раздел №2 - Исполнительная документация
Включает в себя:
- Реестр исполнительной документации
- АООК или АОСР (выбирается инженером)
- Исполнительная схема
- Ведомость объемов работ
- Акт приемочного контроля качества дорожной разметки
- Реестр №1 применяемых материалов

## Раздел №3 - Фактическое выполнение
Таблица с данными выполнения работ на объекте:
- Дата
- Наименование застройщика/объекта
- Материал выполнения (точковка, ТП, ХП, Краска)
- Состав бригады
- Тип линии/Вид работ
- Ширина линии
- п.м./шт
- м2 (вычисляется по формуле: п.м./шт * ширина линии)

## Раздел №5 - Справочник данных
Содержит данные по ширинам линий и их точным расчетам по ГОСТ для правильного вычисления объемов в м2.

## Раздел №6 - Справочник материалов на складе
Информация о материалах:
- Паспорт изделия (номер партии, дата изготовления, срок годности)
- Сертификат/декларация соответствия
- Акт отбора пробы
- Протокол лабораторных испытаний

## Раздел №7 - Контроль
Мониторинг объема остатков продукции с уведомлениями при низком уровне (примерно 20% от объема партии).
